## Trying it out

> git clone https://github.com/ggeorgovassilis/codename-vincent
> cd codename-vincent
> mkdir volumes
> docker-compose up

Add these entries to your /etc/hosts
127.0.0.1     gitlab.cnvc
127.0.0.1     jenkins.cnvc

Visit http://gitlab.cnvc and http://jenkins.cnvc with a browser
